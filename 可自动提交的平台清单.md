## 📝 如何使用自动提交脚本

### 1. 安装依赖
```bash
pip install requests python-dotenv tweepy praw
```

### 2. 创建.env文件
```bash
cp .env.example .env
# 编辑.env文件，填入你的API凭证
```

### 3. 运行脚本
```bash
python auto-submit-with-api.py
```

## 💡 获取API凭证技巧

### 快速获取凭证的平台：
1. **Dev.to** - 登录后直接在设置中生成
2. **GitHub** - 你已有账号，直接生成token

### 需要审核的平台：
1. **Twitter API** - 需要申请开发者账号
2. **LinkedIn API** - 需要创建应用
3. **Product Hunt** - 需要OAuth应用审核

## 🔐 安全提示

3. **定期轮换** API密钥
4. **设置权限** 只给必要的权限
5. **监控使用** 检查API调用量
